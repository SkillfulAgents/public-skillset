Stop staring at the "create new agent" screen. Agent Pill is an onboarding concierge that figures out the single most useful first agent for *you* — and builds it.

**What it does:**

- **Interviews you like a sharp friend, not a survey**

  A short, warm conversation to find the one chore in your week that, if it disappeared, you'd text someone about.

- **Snoops your connected tools (with permission)**

  Peeks at your Gmail, Calendar, Notion, Slack, Linear — whatever you've connected — to spot patterns you wouldn't think to mention.

- **Picks one high-impact agent, not ten mediocre ones**

  Recommends a specific, opinionated first agent tailored to your actual work. No "summarize my email" filler.

- **Builds it for you, then hands you the prompt**

  Creates the agent in your workspace *and* gives you a copyable prompt you can paste anywhere else.

**Smart handoff:** once your first agent is created, Agent Pill steps aside. Follow-up sessions treat you like the experienced user you now are.
